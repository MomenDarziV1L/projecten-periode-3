Een nieuwe repository maken

U kunt een nieuwe repository maken op uw persoonlijke account of een organisatie waar u voldoende rechten voor heeft.

  1. Klik in de rechterbovenhoek van een pagina op (+), en klik vervolgens op (New repository).



  2. Selecteer in de vervolgkeuzelijst (Owner) het account waarop u de repository wilt maken.



  3. Typ een naam voor uw repository en een optionele beschrijving.



  4. Kies ervoor om de repository openbaar of privé te maken. Openbare opslagplaatsen zijn zichtbaar voor het publiek, terwijl privé-opslagplaatsen alleen toegankelijk zijn voor jou en de mensen met wie je ze deelt



  5. Er zijn een aantal optionele items waarmee u uw repository vooraf kunt vullen. Als je een bestaande repository importeert naar GitHub Enterprise, kies dan geen van deze opties, omdat je een samenvoegingsconflict kunt introduceren. U kunt ervoor kiezen om later nieuwe bestanden toe te voegen via de opdrachtregel.

    • U kunt een README maken, een document dat uw project beschrijft.

    • U kunt een .gitignore-bestand maken, een set negeerregels.



  6. Als je klaar bent, klik je op (Create repository).


Een bestand toevoegen aan een repository

Je kunt een bestaand bestand uploaden en vastleggen in een GitHub-repository. Sleep een bestand naar een willekeurige directory in de bestandsstructuur en zet het neer, of upload bestanden vanaf de hoofdpagina van de repository

  1. Navigeer op GitHub naar de hoofdpagina van de repository.



  2. Klik onder de naam van uw repository op Upload files.



  3. Sleep het bestand of de map die je wilt uploaden naar je repository naar de bestandsboom.



  4. At the bottom of the page, type a short, meaningful commit message that describes the change you made to the file. You can attribute the commit to more than one author in the commit message



  5. Bepaal onder de velden voor het vastleggen van berichten of u uw vastlegging aan de huidige tak of aan een nieuwe tak wilt toevoegen. Als je huidige branch master is, zou je ervoor moeten kiezen om een nieuwe branch voor je commit aan te maken en dan een pull request te maken.



  6. Klik op Commit changes.


Een bestand toevoegen aan een opslagplaats via de command line

Je kunt een bestaand bestand uploaden naar een GitHub repository met behulp van de command line.

Waarschuwing: Git nooit gevoelige informatie toevoegen, vastleggen of pushen naar een externe repository. Gevoelige informatie kan omvatten, maar is niet beperkt tot:

  • Passwords
  • SSH keys
  • AWS access keys
  • API keys
  • Credit card numbers
  • PIN numbers

  1. Verplaats op uw computer het bestand dat u naar GitHub wilt uploaden naar de lokale map die is gemaakt toen u de repository kloneerde.



  2. Open Git Bash.



  3. Wijzig de huidige werkmap naar uw lokale repository.



  4. Fase het bestand voor vastlegging in uw lokale repository.

    $ git add .
    # Adds the file to your local repository and stages it for commit. To unstage a file, use 'git reset HEAD YOUR-FILE'.



  5. Leg het bestand vast dat u in uw lokale repository hebt opgeslagen.

    $ git commit -m "Add existing file"
    # Commits the tracked changes and prepares them to be pushed to a remote repository. To remove this commit and modify the file, use 'git reset --soft HEAD~1' and commit and add the file again.



  6. Push de veranderingen in je lokale repository naar GitHub.

    $ git push origin your-branch
    # Pushes the changes in your local repository up to the remote repository you specified as the origin



Wat doe ik in mijn dagelijkse leven


Ik vind deze periode erg moeilijk vanwege de corona-crisis en de school is gesloten en de beperkingen op beweging en vergaderingen.

Dit is mijn dagelijkse routine, als ik werk heb, word ik vroeg om 5 of 6 uur wakker, mijn werk is om pakketten of meubels af te leveren, dus ik moet contact opnemen met de klanten.

Maar vanwege de corona-crisis hebben we nieuwe regels gekregen van onze werkgever, we hoeven alleen de pakketten bij de deur af te leveren en we hoeven niet langer de handtekening van de klant te krijgen, in plaats van dat we een foto als bewijs dat we de pakketten hebben afgeleverd, en we moeten altijd een afstand van anderhalve meter aanhouden.

Voor mijn studie volg ik minder online en probeer ik altijd mijn huiswerk te maken voor de deadline.
Ik heb het gevoel dat online lessen niet veel op school lijken, omdat je je niet genoeg op de leraar kunt concentreren tijdens online lessen vanwege het gedrag van studenten De docenten zijn goed en doen hun best om de studenten te laten profiteren en het beste te geven.

Dus ik leef een normaal leven, ik doe boodschappen en ik ga naar vrienden en zij komen naar mij toe maar zeker met voorzichtiger dat we elkaar niet aanraken en afstand houden.

Ik hoop dat alle patiënten snel herstellen en dat deze crisis snel voorbij is..
